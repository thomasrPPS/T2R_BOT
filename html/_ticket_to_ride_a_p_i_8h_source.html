132 /* ----------------------------------------------------------------
133  * Wait for a T2R Game, and retrieve its name and first data
134  * (the number of cities and the number of connections)
135  *
136  * Parameters:
137  * - gameType: string (max 200 characters) type of the game we want to play
138  * (empty string for regular game)
139  * "TRAINING xxxx" to play with the bot xxxx
140  * "TOURNAMENT xxxx" to join the tournament xxxx
141  * gameType can also contains extra data in form "key1=value1 key2=value1 ..."
142  * to provide options (to bots)
143  * invalid keys are ignored, invalid values leads to error
144  * the options are:
145  * - 'timeout': allows an define the timeout when training (in seconds)
146  * - 'seed': allows to set the seed of the random generator
147  * - 'start': allows to set who starts ('0' to begin, '1' otherwise)
148  * - 'map': allows to choose a map ('USA' for the moment)
149  * the following bots are available:
150  * - DO_NOTHING (stupid player what withdraw cards)
151  *
152  * - gameName: char* to get the game Name (should be allocated, max 50 characters),
153  *
154  * - nbCities: to get the number of cities
155  * - nbTracks: to get the number of tracks between the cities
156  */

160 /* ------------------------------------------------------------

162  * the three arrays are filled by the function
163  *
164  * Parameters:
165  * - tracks: array of (5 x number of tracks) integers
166  * Five integers are used to define a track:

169  * - (3) length of the track (between 1 and 6)

172  * - faceUp: array of 5 t_color giving the 5 face up cards
173  * - cards: array of 4 t_colors with the initial cards in your hand
174  *
175  * (the pointers data MUST HAVE allocated with the right size !!)
176  *
177  * Returns 0 if you begin, or 1 if the opponent begins
178  */
